Privacy Policy

Extension: eledele

Effective date: 9/7/2025

1. Single purpose

This extension has a single, narrow purpose: to remove pesky elements. It does not perform unrelated functions.

2. Summary of data practices

Data collection: This extension does not collect or transmit any personal data. It does not access or store user names, email addresses, payment information, health data, or location data.

Local storage: The extension may store minimal configuration and preference values locally using the browser's storage APIs (for example: preferred settings, UI choices). This data remains on the user's device and is not sent to any external server.

3. Permissions and justification

  • activeTab — Temporarily used when a user activates the extension so it can operate on the current page the user selected.
  • tabs — Used only to identify or update the active tab as required by the extension's single purpose.
  • storage — Used to save lightweight user preferences locally (no personal data).
  • scripting — Used to inject or execute scripts only on pages where the user has explicitly requested the extension's feature.
  • Host permissions — If the extension requests access to specific domains, it is strictly to enable its core functionality on those sites and the scope is limited to the minimum required match patterns.

4. Remote code

This extension does not load or execute remote JavaScript or WebAssembly at runtime. All code executed by the extension is bundled inside the extension package. If you do include external resources in the future, they will be noted here and reviewed for policy compliance.

5. Third parties and data sharing

The extension does not send user data to third parties, nor does it sell or transfer user data. No analytics or advertising SDKs that harvest user data are included.

6. Security

Only the minimal permissions needed to deliver the extension's single purpose are requested. Local data is stored using the browser's secure storage APIs. If any security incidents affecting user data occur, we will follow applicable disclosure and remedial procedures.
